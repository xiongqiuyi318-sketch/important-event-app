# 浏览器通知设置指南

## 如何启用通知权限

### 方法 1：通过应用内提示启用（推荐）

1. 打开应用后，页面顶部会显示一个紫色的通知提示框
2. 点击 **"启用通知"** 按钮
3. 浏览器会弹出权限请求对话框
4. 点击 **"允许"** 或 **"Allow"**

### 方法 2：通过浏览器地址栏启用

#### Chrome / Edge 浏览器：

1. 打开应用 `http://localhost:5173`
2. 点击地址栏左侧的 **🔒 锁图标** 或 **ℹ️ 信息图标**
3. 在弹出菜单中找到 **"通知"** 或 **"Notifications"**
4. 选择 **"允许"** 或 **"Allow"**
5. 刷新页面（按 F5）

#### Firefox 浏览器：

1. 打开应用 `http://localhost:5173`
2. 点击地址栏左侧的 **锁图标**
3. 点击右侧的 **>** 箭头
4. 找到 **"接收通知"** 权限
5. 选择 **"允许"**
6. 刷新页面

### 方法 3：通过浏览器设置启用

#### Chrome / Edge：

1. 点击浏览器右上角的 **三点图标 ⋮**
2. 选择 **"设置"**
3. 左侧菜单选择 **"隐私和安全"**
4. 点击 **"网站设置"**
5. 点击 **"通知"**
6. 在 **"允许"** 部分，点击 **"添加"**
7. 输入：`http://localhost:5173`
8. 点击 **"添加"**

#### Firefox：

1. 点击浏览器右上角的 **三条横线图标 ≡**
2. 选择 **"设置"**
3. 左侧选择 **"隐私与安全"**
4. 滚动到 **"权限"** 部分
5. 找到 **"通知"**，点击 **"设置"**
6. 搜索 `localhost:5173`
7. 将状态改为 **"允许"**
8. 点击 **"保存更改"**

## 测试通知

启用通知后：

1. 应用会立即显示一个测试通知：**"通知已启用 ✓"**
2. 创建一个事件并设置提醒时间
3. 到达提醒时间时，您会收到推送通知
4. **即使最小化浏览器，也能收到通知！**

## 通知功能说明

### ✨ 新功能特点：

- 🔔 **后台通知**：即使最小化浏览器也能收到提醒
- ⏰ **自动提醒**：到达设置的时间自动弹出通知
- ⚠️ **过期标识**：过期的提醒会在通知中显示 "⚠️ 已过期"
- 📳 **振动提示**：支持设备振动（移动设备）
- 🖱️ **点击聚焦**：点击通知可以回到应用窗口

### 🎨 UI 更新：

- ✅ **已过期步骤**：显示为红色字体，带 ⚠️ 标记
- ✅ **开始时间过期**：事件标题显示为红色
- ✅ **截止时间过期**：事件卡片红色背景，标题和时间为红色

## 常见问题

### Q: 我点了"阻止"，如何重新启用？

A: 使用上面的"方法 2"或"方法 3"手动在浏览器设置中启用。

### Q: 为什么我没收到通知？

A: 检查：
1. 通知权限是否已允许
2. 浏览器标签页是否被关闭（需要保持打开或最小化）
3. 系统通知是否被禁用（Windows 通知设置）
4. 浏览器是否在运行

### Q: 通知能在手机上使用吗？

A: 可以！在手机浏览器中打开应用，同样可以启用推送通知。

### Q: 关闭浏览器后还能收到通知吗？

A: 不能。需要浏览器保持运行（可以最小化），但标签页可以在后台。
