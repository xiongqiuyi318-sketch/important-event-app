# 重要事件备忘录

一个基于React的重要事件管理应用，采用四象限优先级视图来组织和管理事件。

## 功能特性

- 📋 **四象限视图**：按优先级（重要且紧急、重要、一般、不重要）组织事件
- ➕ **事件管理**：新增、编辑、删除事件
- 📝 **智能步骤生成**：根据事件标题、分类和描述自动生成完成步骤
- ✅ **步骤跟踪**：标记完成步骤，自动更新事件状态
- 🔄 **手动排序**：在象限内上下移动事件顺序
- 📅 **时间管理**：设置开始时间和截止时间（Deadline）
- 🏷️ **分类系统**：支持9种事件分类（发货、进口、本地销售等）
- 📜 **历史记录**：查看已完成或过期的事件
- 💾 **本地存储**：数据保存在浏览器localStorage中

## 特殊功能

### 发货分类智能步骤
当选择"发货"分类时，系统会自动生成以下步骤：
1. 客人确认block list
2. 货代配柜+制作T2L
3. 发货+制作delivery note
4. 制作invoice并发给客人+货代+会计师
5. 货代给我MRN和VGM
6. 最终的完整单证给到客人

这些步骤可以手动调整和修改。

## 使用说明

1. **新增事件**：点击"新增事件"按钮，填写事件信息
   - 输入事件标题和描述后，系统会自动生成步骤
   - 可以手动添加、删除步骤
   - 选择事件分类和优先级

2. **管理步骤**：
   - 勾选复选框标记步骤完成
   - 完成的步骤会显示删除线
   - 可以在事件卡片中直接添加新步骤

3. **调整优先级**：编辑事件时可以更改优先级，事件会自动移动到对应象限

4. **手动排序**：在同一象限内，使用上下箭头按钮调整事件顺序

5. **查看历史**：点击"历史"标签查看已完成或过期的事件
