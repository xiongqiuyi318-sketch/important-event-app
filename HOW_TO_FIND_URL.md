# 如何查询应用的访问链接

## 📍 方法一：在 Vercel 控制台查看（推荐）

### 步骤：

1. **登录 Vercel**
   - 访问：https://vercel.com/login
   - 使用 GitHub 账号登录

2. **进入项目列表**
   - 登录后会自动进入 Dashboard（控制台）
   - 或者点击左上角的 Vercel Logo 返回首页

3. **找到你的项目**
   - 在项目列表中找到 `important-event-app` 项目
   - 点击项目名称进入项目详情页

4. **查看访问链接**
   - 在项目详情页的顶部，你会看到：
     - **"Visit" 按钮**：点击可直接访问应用
     - **访问链接**：`https://important-event-app.vercel.app`
   - 在 "Production Deployment" 部分的 "Domains" 区域也会显示访问链接

### 你的应用访问链接：

**主链接：** `https://important-event-app.vercel.app`

---

## 📍 方法二：在项目卡片上查看

在 Vercel Dashboard 的项目列表中，每个项目卡片上都会显示：
- 项目名称
- **访问链接**（蓝色链接，格式：`项目名.vercel.app`）
- 直接点击链接即可访问

---

## 📍 方法三：通过部署记录查看

1. 进入项目详情页
2. 点击左侧菜单的 **"Deployments"**（部署）
3. 在部署列表中，每个部署都有一个访问链接
4. 找到状态为 **"Ready"** 的生产环境部署
5. 点击部署卡片上的链接即可访问
